## ⚙️ Run Locally

Clone the project

```bash
git clone https://github.com/DEMYSTIF/go-ransomware.git
cd go-ransomware
```

Build binary file

```bash
go build voldemort.go
```

Encrypt files in the folder

```bash
./voldemort avada kedavra
```

Decrypt files in the folder

```bash
./voldemort expecto patronum
```
